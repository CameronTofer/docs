---
title: "send_action"
---

Send an action element to the chatbot UI.
If a project ID is configured, the messages will be uploaded to the cloud storage.

### Parameters
<ParamField path="name" type="str">
    The name of the action to send.
</ParamField>
<ParamField path="trigger" type="str">
  The text that should trigger the action when clicked.
</ParamField>
<ParamField path="description" type="str" optional>
The description of the action. Defaults to "".
</ParamField>


<RequestExample>
```python Code Example
import chainlit as cl

@cl.action("action1")
def on_action(action):
    cl.send_message("Executed action 1!")

@cl.on_chat_start
def main():
    cl.send_message("Hello, here is a clickable action!")
    cl.send_action(name="action1", trigger="clickable action",
                   description="Click on this to run action1!")
```
</RequestExample>
