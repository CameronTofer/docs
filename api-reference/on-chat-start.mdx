---
title: "on_chat_start"
---

Hook to react to the user websocket connection event.

### Parameters
<ParamField path="user_env" type="Dict[str, str]">
  The user env variables if defined in the config file, else an empty dict.
</ParamField>

<RequestExample>
```python
from chainlit import send_message, ask_user, on_chat_start

bot_name = "Example"


@on_chat_start
def main(env):
    res = ask_user(author=bot_name, content="What is your name?", timeout=10)
    if res:
        send_message(
            author=bot_name,
            content=f"Your name is: {res['content']}",
        )
```
</RequestExample>
