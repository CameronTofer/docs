---
title: "on_chat_start"
---

Hook to react to the user websocket connection event.

<RequestExample>
```python Code Example
from chainlit import send_message, ask_for_input, on_chat_start


@on_chat_start
def main():
    res = ask_for_input(content="What is your name?")
    if res:
        send_message(
            content=f"Your name is: {res['content']}",
        )
```
</RequestExample>
