---
title: "on_message"
---

Framework agnostic decorator to react to messages coming from the UI.
The decorated function is called every time a new message is received.

### Parameters

<ParamField path="message" type="str">
  The message coming from the UI.
</ParamField>

<ParamField path="user_env" type="Dict[str, str]">
  The user env variables if defined in the config file, else an empty dict.
</ParamField>

<RequestExample>
```python
from chainlit import on_message

@on_message
def main(message: str, user_env):
  # do something
```
</RequestExample>
