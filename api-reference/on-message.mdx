---
title: "@on_message"
---

Framework agnostic decorator to react to messages coming from the UI.
The decorated function is called every time a new message is received.

### Parameters

<ParamField path="func" type="Callable[[str], Any]">
  The function to be called when a new message is received.
  The function takes the string message as a parameter.
</ParamField>

### Returns

<ResponseField name="on_message" type="Callable[[str], Any]" required>
  The decorated on_message function.
</ResponseField>

<RequestExample>
```python
from chainlit import @on_message

@on_message
def main(message: str):
  # do something
```
</RequestExample>
