---
title: "send_token"
---

This function sends a token element to the chatbot UI. For an in-depth example: [@streaming](/concepts/streaming)

### Parameters

<ParamField path="token" type="str">
  The content of the token element
</ParamField>

### Usage

```python Code Example
import chainlit as cl

msg = ""
cl.start_stream()

# Send a list of token elements

token_list = ["the", "quick", "brown", "fox"]
for token in token_list:
cl.send_token(token)
msg += token + " "

# End the stream and send the final message

cl.send_message(content=msg, end_stream=True)

```
