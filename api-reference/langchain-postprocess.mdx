---
title: "langchain_postprocess"
---

This function is useful to post-process the response of a LangChain object instantiated with the [@langchain_factory](/api-reference/langchain-factory) decorator. The decorated function takes the raw output of the LangChain object and returns a string as the final response.

### Parameters

<ParamField path="res" type="Any">
  The raw response of the langchain agent
</ParamField>

<ParamField path="user_env" type="Dict[str, str]">
  The user env variables if defined in the config file, else an empty dict.
</ParamField>

### Returns

<ResponseField name="response" type="str" required>
  The final response string.
</ResponseField>

<RequestExample>
```python
from langchain import OpenAI, LLMMathChain
from chainlit import langchain_factory, langchain_postprocess

@langchain_factory
def load(user_env):
    llm = OpenAI(temperature=0)
    llm_math = LLMMathChain(llm=llm)
    return llm_math

@langchain_postprocess
def post_process(llm_math_res, user_env):
    response = # do something with llm_math_res
    return response
```
</RequestExample>
