Sometimes, you may want to make your app more interactive by allowing users to perform actions. For example, you may want to allow users to generate a variant of an image your app previously generated.

<Card title="Action API" icon="link" href="/api-reference/action">
  Learn how to send actions to the user interface.
</Card>

Chainlit lets you send clickable buttons to the user interface and declare callbacks for when the user clicks on them.
