Chainlit Cloud allows you to:

- Add authentication to your application
- Add team members with different roles
- Persist the conversations and feedback of your team members
- Deploy your application with a single line (coming soon)
- Run non regression tests (coming soon)

## Create a project

Head to [Chainlit Cloud](https://cloud.chainlit.io) and login.
Then, create a project by clicking the + icon. You should see something like this:
![Project](/images/project.png)

<Note>You can copy the project ID by clicking on it.</Note>
