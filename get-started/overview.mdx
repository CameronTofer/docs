Chainlit is an open-source Python package that makes it incredibly fast to build and share LLM apps. Integrate the Chainlit API in your existing code to spawn a ChatGPT-like interface in minutes!

## Key features

1. [Build LLM Apps fast:](/examples/openai-sql) Integrate seamlessly with an existing code base or start from scratch in minutes

2. [Collect user feedback:](/chat-data/feedback) Invite your teammates, create annotated datasets and run experiments together

3. [Visualize multi-steps reasoning:](/observability-iteration/chain-of-thought) Understand the intermediary steps that produced an output at a glance

4. [Iterate on prompts:](/observability-iteration/prompt-playground) Deep dive into prompts in the Prompt Playground to understand where things went wrong and iterate
