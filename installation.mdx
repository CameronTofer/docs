---
title: Installation
---

Chainlit requires `python>=3.7`.


<Warning>The Chainlit package has not yes been publicly released.</Warning>
You can install the package with:
```bash
pip install chainlit
```

To try if everything works correctly run:
```bash
chainlit hello
```

This should spawn the chainlit UI and ask for your name like so:
![Hello](/images/hello.png)
