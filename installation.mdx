---
title: Installation
---

Chainlit requires `python>=3.8`.

Open a terminal and run:
```bash
pip install chainlit
```

Make sure everything runs smoothly:
```bash
chainlit hello
```

This should spawn the chainlit UI and ask for your name like so:
![Hello](/images/hello.png)
