Chainlit is an open-source Python package that makes it incredibly fast to build and share LLM apps. Integrate the Chainlit API in your existing code to spawn a ChatGPT-like interface in minutes!

<Frame caption="Demo of the image gen app available in the chainlit cookbook repo.">
  <img src="/images/overview.gif" />
</Frame>

## Key features

1. [Build LLM Apps fast:](/examples/openai-sql) Integrate seamlessly with an existing code base or start from scratch in minutes

2. [Visualize multi-steps reasoning:](/concepts/chain-of-thought) Understand the intermediary steps that produced an output at a glance

3. [Iterate on prompts:](/concepts/prompt-playground) Deep dive into prompts in the Prompt Playground to understand where things went wrong and iterate

4. [Collaborate with teammates:](/cloud/create-project) Invite your teammates, create annotated datasets and run experiments together

5. [Share your app:](/cloud/deploy) Publish your LLM app and share it with the world (coming soon)

## Integrations

Chainlit is compatible with all Python programs and libraries. That being said, it comes with a set of integrations with popular libraries and frameworks.
