---
title: With LangChain
---

Chainlit seamlessly integrates with LangChain to create sharable apps in seconds.

```python app.py
from langchain import PromptTemplate, OpenAI, LLMChain
import chainlit as cl
import os

os.environ["OPENAI_API_KEY"] = "YOUR_OPEN_AI_API_KEY"

template = """Question: {question}

Answer: Let's think step by step."""

@cl.langchain_factory
def factory():
    prompt = PromptTemplate(template=template, input_variables=["question"])
    llm_chain = LLMChain(prompt=prompt, llm=OpenAI(temperature=0), verbose=True)

    return llm_chain
```

Wrap the logic that instantiate the LangChain agent in a function and add the [@langchain_factory](/api-reference/langchain-factory) decorator.

That's it. Now just run:

```bash
$ chainlit run app.py -w
```
<Warning>When using LangChain, caching is turned on by default! The cache lives in the .chainlit folder.</Warning>
