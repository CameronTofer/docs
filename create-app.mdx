## Basics

Chainlit can turn your python code into a chatbot in a few lines of code.

```python
from chainlit import send_message, on_message

@on_message # this function will be called every time a user inputs a message in the UI
def main(message: str):

    # custom logic...

    # send back a reply to the user
    send_message(
      author="Test",
      content=f"Received: {message}",
    )
```

## The App

Let's build a simple app that helps users to create SQL queries with natural language.

### Imports

<Warning>
  It is not a good practice to hard code environment variables. For a better
  way, check the [Chainlit config](/cloud/config).
</Warning>

```python app.py
from chainlit import on_message, send_message, LLMSettings
import openai

openai.api_key = "YOUR_OPEN_AI_API_KEY"
```

### Define a prompt and LLM settings

````python app.py
prompt = """SQL tables (and columns):
* Customers(customer_id, signup_date)
* Streaming(customer_id, video_id, watch_date, watch_minutes)

A well-written SQL query that {input}:
```"""

model_name = "text-davinci-003"

settings = {
    "temperature": 0,
    "max_tokens": 500,
    "top_p": 1,
    "frequency_penalty": 0,
    "presence_penalty": 0,
    "stop": ["```"]
}
````

### Listen and Reply

Here, we decorate the `main` function with the [@on_message](api-reference/on-message) decorator to tell Chainlit to run the `main` function each time a user sends a message.
Then, we send back the answer to the UI with the [send_message](api-reference/send-message) function.

```python app.py
@on_message
def main(message: str):
    fromatted_prompt = prompt.format(input=message)
    response = openai.Completion.create(
        model=model_name,
        prompt=fromatted_prompt,
        **settings
    )
    content = response["choices"][0]["text"]

    send_message(
        author="SQL chatbot",
        language="sql",
        content=content,
        prompt=fromatted_prompt,
        llm_settings=LLMSettings(model_name=model_name, **settings)
    )
```

### Run the UI

```bash
chainlit run app.py
```

You can also automatically reload the UI on file change with the `-w` flag like so:
```bash
chainlit run app.py -w
```
