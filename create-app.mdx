For this example, we are going to build a simple app that helps users to create SQL queries with natural language using the `openai` python package.

## Original app code

<Warning>It is not a good practice to hard code environment variables. For a better way, check the [Chainlit config](/cloud/config).</Warning>

```python app.py
import openai

openai.api_key = "YOUR_OPEN_AI_API_KEY"

prompt = """SQL tables (and columns):
* Customers(customer_id, signup_date)
* Streaming(customer_id, video_id, watch_date, watch_minutes)

A well-written SQL query that {input}:
```"""

model_name = "text-davinci-003"

settings = {
    "temperature": 0,
    "max_tokens": 500,
    "top_p": 1,
    "frequency_penalty": 0,
    "presence_penalty": 0,
    "stop": ["```"]
}

def main(message: str):
    fromatted_prompt = prompt.format(input=message)
    response = openai.Completion.create(
        model=model_name,
        prompt=fromatted_prompt,
        **settings  
    )
    content = response["choices"][0]["text"]
    return content

message = "Compute the number of customers that watched more than 50 minutes of video"
print(main(message))
```

## With Chainlit
While functional, this program makes it hard for you to ask multiple questions, and reading answers in a terminal or jupyter notebook is not ideal.
Let's see how you can spawn a UI with minimal changes.

```python app.py
import openai
from chainlit import on_message, send_message, LLMSettings

openai.api_key = "YOUR_OPEN_AI_API_KEY"

prompt = """SQL tables (and columns):
* Customers(customer_id, signup_date)
* Streaming(customer_id, video_id, watch_date, watch_minutes)

A well-written SQL query that {input}:
```"""

model_name = "text-davinci-003"

settings = {
    "temperature": 0,
    "max_tokens": 500,
    "top_p": 1,
    "frequency_penalty": 0,
    "presence_penalty": 0,
    "stop": ["```"]
}

@on_message
def main(message: str):
    fromatted_prompt = prompt.format(input=message)
    response = openai.Completion.create(
        model=model_name,
        prompt=fromatted_prompt,
        **settings  
    )
    content = response["choices"][0]["text"]

    send_message(
        author="SQL chatbot",
        language="sql",
        content=content,
        prompt=fromatted_prompt,
        final=True, 
        llm_settings=LLMSettings(model_name=model_name, **settings)
    )
```
Here, we decorated our `main` function with the `@on_message` decorator to tell Chainlit to run the `main` function each time a user sends a message.
Then, we send back the answer to the UI with the `send_message` function.

To spawn the UI, simply run:
```bash
chainlit run app.py
```
